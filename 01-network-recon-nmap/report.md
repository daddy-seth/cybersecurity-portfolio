# Network Reconnaissance with Nmap

## Objective
Learn how to discover active hosts on a network.

## Tool Used
- Kali Linux
- Nmap

## Command
nmap -sn 10.0.2.0/24

## Results
- 10.0.2.2 (VirtualBox NAT Gateway)
- 10.0.2.3 (VirtualBox Service)
- 10.0.2.15 (My Kali Machine)
